## TEMPORAL CORE FABRIC (pogls_temporal_core.h)

```
cycle = 720 steps
phase = step % 720
core_id   = phase % N         (which virtual core)
core_tick = phase / N         (run only when == 0)

→ each core runs 1 time per 720-step cycle
→ N=4: each core runs at phase 0,1,2,3

Sub-cycle events:
  phase % 40  = TC_EVENT_BURST
  phase % 60  = TC_EVENT_GEAR_LIGHT
  phase % 80  = TC_EVENT_GEAR_MED
  phase % 144 = TC_EVENT_ANCHOR  ← strong enforcement
  phase == 719 = TC_EVENT_CYCLE_END

Per-core:
  - isolated state_hash (XOR rolling)
  - rewind buffer (18 slots = gate_18)
  - zero lock, zero contention
```
