⬡ QRPN Core — What It Actually Does
A = radial(value, N, seedA) ← World A projection
B = radial(value, N, seedB) ← World B projection
────────────────────────────────
c² = A² + B² ← Pythagorean generator
a²+b² = (A²-B²)² + (2AB)² ← self-check (always = c²)
────────────────────────────────
Cq = mix32(c) ← CPU signature
Cg = phi_scatter_hex(value) ← GPU witness (hex fold)
────────────────────────────────
PASS if: (A+B == Cg) OR (Cq == Cg)
FAIL if: BOTH mismatch → shadow_fail++
