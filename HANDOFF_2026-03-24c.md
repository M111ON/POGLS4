## ROOT CAUSE (fixed this session)

**Anchor skew 5-0-0-0** — TC_EVENT_ANCHOR fires at
`phase % 144 == 0` → phases 0,144,288,432,576 → all `core_id = phase%4 = 0`
→ core[0] got ALL anchor updates.

**Fix:** `ma_step()` now iterates ALL cores on every ANCHOR event.

## 5 PRODUCTION FIXES APPLIED

### FIX 1 — Anchor timing hash-mix (was: alias pattern)
```c
// BEFORE (aliasing — many core_ids map same phase bucket)
fire_phase = (TC_ANCHOR + ci * 37u) % TC_ANCHOR;

// AFTER (hash-mix — uniform, no aliases)
uint32_t h = (phase ^ (ci * 0x9E3779B1u)) % TC_ANCHOR;
if (h == 0 && ((step_idx + ci) & 1u) == 0u)   // parity guard
```

### FIX 2 — Step parity guard
`((step + core_id) & 1) == 0` → halves fire rate, removes lucky-hit bias.

### FIX 3 — Signed noise [-8,+7] (was: +[0,3])
```c
// BEFORE — too weak to break local minima
noise = hash(...) & 3u;           // range [0,3]

// AFTER — strong enough, still deterministic
int32_t noise = (int32_t)((hash(core_id, step, i) & 15u)) - 8;
penalized = (uint64_t)((int64_t)penalized + noise);
```

### FIX 4 — Bias decay 99% → 95%
```c
// BEFORE: memory depth ~100 anchor events (too long → stuck)
bias[b] = bias[b] * 99u / 100u;

// AFTER: memory depth ~20 anchor events (responsive)
bias[b] = bias[b] * 95u / 100u;
```

### FIX 5 — Adaptive bias_k: clamp + corrected thresholds
```c
if (rate_pct > 75u) bias_k--;    // too jittery
if (rate_pct < 10u) bias_k++;    // too stuck
// hard clamp
if (bias_k < 1u) bias_k = 1u;
if (bias_k > 4u) bias_k = 4u;
```
